RaspiFM
=======

FM modulator for the Raspberry Pi in C, with Python script for playing multiple files.

The C code will be the bits from:
http://www.icrobotics.co.uk/wiki/index.php/Turning_the_Raspberry_Pi_Into_an_FM_Transmitter

However the Python code will be my own, since everyone else seems to think it is sufficient to play one file.
